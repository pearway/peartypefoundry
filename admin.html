폰트 라이브러리

사이트의 폰트 카드로 표시되는 항목입니다. 위에서 아래 순서로 노출됩니다.

폰트 클릭 시 상세 페이지로 이동 (font.html?id=...)

무료 폰트: 가격을 Free 또는 무료로 입력 + 다운로드 파일 경로 입력 → 직접 다운로드 버튼
유료 폰트: 가격을 다른 값으로 입력 + 외부 구매 URL 입력 → 구매처 링크 버튼 (눈누, 산돌 등)

이미지/시안/ZIP은 GitHub에 직접 업로드. 어드민에선 경로만 입력합니다.
스타일 — serif (이태릭) / display (skew) / default (일반) · 테마 — light / dark / dim

그래픽 작업

사이트의 그래픽 카드로 표시되는 작업물입니다.

그래픽 카드 클릭 시 라이트박스로 이미지 확대. 별도 상세 페이지 없습니다.
이미지 경로 입력 시 라이트박스에 그 이미지가 표시되고, 비워두면 미리보기 텍스트가 큰 화면으로 표시됩니다.

인사이트 챕터

Font Insight 시리즈 — 카드뉴스/PDF 콘텐츠 챕터를 관리합니다.

워크플로우
1. PDF 파일을 GitHub pdfs/ 폴더에 업로드 (예: pdfs/insight-01.pdf)
2. 표지 이미지(PDF 첫 페이지 캡처)를 images/insight/에 업로드
3. 여기서 챕터 정보 입력 + 두 파일의 경로 입력
4. data.json 다운로드 → GitHub에 푸시

본문 글: HTML 입력 가능. 비워두면 표지 + 다운로드만 표시됩니다.

배포 가이드

현재 입력된 데이터의 JSON 형태입니다.

배포 워크플로우
  1. 이미지 파일을 GitHub 저장소의 images/ 폴더에 직접 업로드
  2. 이 어드민에서 콘텐츠 입력 + 이미지 경로 작성
  3. 상단 "data.json 다운로드" 클릭
  4. 다운받은 data.json을 GitHub 저장소 루트에 업로드 (덮어쓰기)
  5. 커밋 → Vercel 자동 배포 (1~2분)
