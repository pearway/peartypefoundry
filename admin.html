이미지 사용법

여기서는 그 경로만 입력하면 됩니다. 예: images/hero.jpg

연락처 / 소셜

폰트 라이브러리

사이트의 폰트 카드로 표시되는 항목입니다. 위에서 아래 순서로 노출됩니다.

대표 이미지 사용 시: images/fonts/elegant-serif.jpg 같은 경로 입력. 비워두면 미리보기 글자가 표시됩니다.
스타일 — serif (이태릭 표시) / display (skew 표시) / default (일반)
테마 — light (베이지 배경) / dark (검정 배경) / dim (어두운 회색, 진행 중)

그래픽 작업

사이트의 그래픽 카드로 표시되는 작업물입니다.

대표 이미지 사용 시: images/works/poster1.jpg 같은 경로 입력. 비워두면 미리보기 텍스트가 표시됩니다.

배포 가이드

현재 입력된 데이터의 JSON 형태입니다.

배포 워크플로우
  1. 이미지 파일을 GitHub 저장소의 images/ 폴더에 직접 업로드
  2. 이 어드민에서 콘텐츠 입력 + 이미지 경로 작성
  3. 상단 "data.json 다운로드" 클릭
  4. 다운받은 data.json을 GitHub 저장소 루트에 업로드 (덮어쓰기)
  5. 커밋 → Vercel 자동 배포 (1~2분)
